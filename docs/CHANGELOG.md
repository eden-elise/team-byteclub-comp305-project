# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-09

### Added
- Floor 1-4 gameplay implementation
- Battle system with boss fights (Cursed Scholar, Lord Dravik)
- Audio system with sound effects and volume control
- Status effect system with visual icons
- TypewriterTextBox dialogue system

### Changed
- Refactored battleScene for reduced complexity
- Updated Dravik difficulty balancing

### Fixed
- Text wrap errors
- Status icon image locations
- Floor transition after boss battles
